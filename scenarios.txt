1. Registration:
- anonymous visits the main page
- clicks register
- fill form
- click register

2. Login
- anonymous visits the main page
- clicks login
- fill login form
- clicks login button

3. Search pets
- user visits main page
- observes the list of all pets
- enters breed or age or color into the search field
- clicks search button

4. Pets introduction
- admin visits main page
- clicks add pet button
- fills form
- clicks add button

5. Pets reservation
- after part 3. Search
- user clicks on pet in a list
- opens window with question about reservation
- user clicks "reserve" or "cancel"
